# Bright-Coffee-Shop-Case-Study
This project analyzes a coffee shop dataset to uncover insights into sales,customer behaviour and product performance.
The goal of this project was to find out which products generate the most revenue,
Which products generate the most revenue,sales trends across products and time intervals and
recommendations for improving sales performance.
The tool used were , Databricks,Excel,Miro and Canva.
This project demonstrates how data analysis can help a coffee shop improve its decision making to optimize sales.
